My Projects

FAERS - Adverse Drug Reactions

This project focuses on large-scale analysis of the FDA Adverse Event Reporting System to uncover drug safety patterns from real-world post-marketing data. I built an end-to-end data pipeline to clean, standardize, and integrate millions of adverse event reports across multiple FAERS tables, including drug information, reported reactions, dosages, and patient demographics. Using SQL-based association analysis, I identified statistically significant drug-symptom relationships and explored potential drug-drug interaction risks. I also conducted dose-response analysis to examine how adverse reactions change with increasing dosage levels, and performed exploratory data analysis to surface meaningful safety insights.

Similar Image Finder

This project focuses on building a scalable image similarity system using OpenAI’s CLIP model and K-Nearest Neighbors (KNN). Trained on over 21,000 images from the Unsplash Lite dataset, the system generates semantic image embeddings and retrieves the top 6 visually and semantically similar images for any input image. By leveraging pretrained vision–language representations, the model achieves strong semantic matching without task-specific training. This project demonstrates my experience with deep learning embeddings, similarity search, and deploying machine learning systems for real-world use cases such as content discovery and media organization.

Recipe Recommendation System

The Recipe Recommendation System is designed to help users discover recipes based on the ingredients they already have. Using a K-Nearest Neighbors (KNN) approach, the system matches user-input ingredients with a structured recipe dataset to identify the most relevant and similar recipes. In addition to direct matches, the model suggests alternative recipes with overlapping ingredient profiles, encouraging exploration and reducing food waste. This project highlights my understanding of recommendation logic, feature representation, and applying machine learning techniques to practical, user-driven problems.
