My Projects

FAERS - Adverse Drug Reactions

Created a comprehensive pharmacovigilance analysis platform for the FDA Adverse Event Reporting System (FAERS), processing over 18 million real-world adverse drug reaction reports spanning multiple years and quarters. The project standardizes nearly 1 million inconsistent drug names, dosages, and reaction terms using Python and SQL, then applies statistical association mining, synergy ratio calculations, and dose-response modeling to uncover significant drug-symptom patterns, interaction risks, and escalation signals at higher dosages. Interactive Plotly dashboards provide dynamic exploration of symptom frequency, interaction networks, dosage effects, and top safety signals, supporting evidence-based post-marketing drug safety monitoring and regulatory insights.

Similar Image Finder

Built a semantic image retrieval system leveraging OpenAI's CLIP (ViT-B/32) model to enable zero-shot visual and conceptual similarity search across a dataset of over 21,000 high-quality Unsplash images. The end-to-end pipeline generates multimodal embeddings in PyTorch, indexes them for efficient querying, and performs K-Nearest Neighbors search with cosine distance in Scikit-Learn, achieving 0.91 mean similarity accuracy across 1,000 evaluation queries. Matplotlib visualizations display side-by-side retrieval results and performance metrics, demonstrating strong semantic understanding for applications in content discovery, media organization, and visual recommendation systems without task-specific training.

Recipe Recommendation System

The Recipe Recommendation System is designed to help users discover recipes based on the ingredients they already have. Using a K-Nearest Neighbors (KNN) approach, the system matches user-input ingredients with a structured recipe dataset to identify the most relevant and similar recipes. In addition to direct matches, the model suggests alternative recipes with overlapping ingredient profiles, encouraging exploration and reducing food waste. This project highlights my understanding of recommendation logic, feature representation, and applying machine learning techniques to practical, user-driven problems.
